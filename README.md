This project demonstrates an **Employee Attendance Analysis Dashboard** built using:
- **Excel**
- **Power Query** for data cleaning and transformation
- **Power Pivot** for data modeling and DAX calculations
- Interactive **Excel dashboard** with KPIs, charts, and slicers

The dashboard provides insights into:
- Total Present & Absent
- Overtime Hours
- Average Check-in Time
- Absenteeism Percentage
- Department-wise and Employee-wise Attendance
- Leave Type Distribution
